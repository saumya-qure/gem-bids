Setup — load session tokens

GeM's session cookies are HttpOnly — JavaScript can't read them. The only reliable way to load tokens is to copy a real request from DevTools' Network tab. It takes ~30 seconds:

  1. Open bidplus.gem.gov.in/all-bids and log in if needed.
  2. On that page, open DevTools (Cmd+Option+I) → click the Network tab.
  3. In the filter box at the top of the Network tab, type all-bids-data.
  4. In the GeM page, change any filter or click "Search" — this fires a real API request. You'll see a row appear in Network.
  5. Right-click that all-bids-data row → Copy → Copy as cURL (on macOS Chrome it may say "Copy as cURL (bash)").
  6. Paste it below and click Load tokens:

Tokens last a few hours. When they expire (Fetch returns 401/403), repeat the same paste with a fresh cURL.
